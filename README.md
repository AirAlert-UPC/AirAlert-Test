## Informe de Implementación: AirAlert

AirAlert es una solución tecnológica diseñada para el monitoreo inteligente y la prevención respiratoria en tiempo real dentro de Lima Metropolitana. El objetivo principal de este proyecto es optimizar la calidad de vida de las personas que padecen de condiciones respiratorias crónicas, brindándoles herramientas para anticipar las drásticas variaciones climáticas y los niveles de polución característicos de la costa limeña.

## Segmento Objetivo
La solución digital está orientada a los siguientes perfiles de usuario:
* Pacientes con diagnósticos de condiciones respiratorias crónicas, tales como asma, alergias, EPOC o sensibilidad pulmonar.
* Cuidadores y familiares que requieran supervisar el entorno ambiental de personas vulnerables para ejecutar medidas preventivas oportunas.

---

## Características Principales del Sistema
* **Monitoreo en Tiempo Real:** Visualización detallada del Índice de Calidad del Aire (AQI) segmentado por distritos de Lima Metropolitana.
* **Alertas Preventivas Automatizadas:** Emisión de notificaciones anticipadas ante el deterioro de las condiciones ambientales para mitigar riesgos de salud.
* **Gestión de Perfiles Familiares:** Incorporación de un modo de cuidador que permite centralizar el monitoreo de múltiples usuarios desde una sola plataforma.
* **Interfaz Adaptativa:** Desarrollo de una landing page optimizada mediante el uso exclusivo de HTML, CSS y JavaScript nativo, asegurando un rendimiento fluido en diversos dispositivos.
